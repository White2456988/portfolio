制作内容

このプロジェクトでは、結婚式をテーマにしたWebデザインを制作しました。

詳細情報

  • ターゲット：結婚式を控えた新婦（主に20〜30代女性）
  • 目的：一目で必要な情報が伝わるよう、サービス案内 → 挙式の流れ → 会場情報 の順に構成。視覚的にわかりやすく設計。
  • こだわり：JavaScriptで雪のエフェクトを実装し、冬の結婚式をイメージした幻想的な雰囲気を演出。
  • 課題：スマートフォンでも美しく見えるようレスポンシブ対応を工夫。アニメーションと読みやすさのバランスに悩んだ。
  • 制作期間：1カ月
  • 使用技術：HTML / CSS / JavaScript（アニメーション / DOM操作）
